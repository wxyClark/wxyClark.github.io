## 命令行
windows/macOS用户可通过官网脚本
```sh
bash -c "$(curl -fsSL https://gitee.com/iflow-ai/iflow-cli/raw/main/install.sh)"
```
或npm包管理器完成部署。启动后通过/model命令选择模型，推荐新手使用Qwen3-Coder或GLM-4.5

登录务必选择"Login with iFlow"方式

关闭命令行再重新开启时，只需在终端直接输入 iflow 命令即可启动
若提示"command not found"，通常是环境变量未持久化，可通过 source ~/.bashrc（Linux/macOS）或重启终端解决；Windows用户需确保Node.js安装路径已添加至系统环境变量。
