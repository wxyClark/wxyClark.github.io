# 道&法

```tip
大道至简

在职场，时间的质量最关键；

在家里，时间的长短最重要。
```

* 读书法：爷爷书 结构 主题阅读 笔记 收入自己的体系 【实践】修正
* 费曼学习法：学 然后知不足，教 然后知困
* 强迫效率法则：时间有限，任何人都无法做完所有的事情
* 二八法则：20%的重要活动贡献80%的成果，罗列(经济、家庭、身体)各3件具性价比的事情，全身心投入。
* 平衡不是可选择的，只能在行动中不断调整。

* 没有思路时，休息一下 —— 切换任务 或 冥想10分钟
