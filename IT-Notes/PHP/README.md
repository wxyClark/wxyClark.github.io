[php-fig WEB](https://www.php-fig.org/)

[php-fig GitHub](https://github.com/php-fig)

phpize是用来扩展PHP扩展模块的，通过phpize可以建立PHP的外挂模块。

比如想在原来编译好的PHP中加入Memcached等扩展模块，可以使用phpize工具。

./configure后面可以指定的是php-config文件的路径。
