# 围棋

```tip
少儿围棋入门使用9路小棋盘
```

## 规则

* 行礼：请多指教
* 猜先、让先
* 黑先白后、交替落子、落子无悔
* 气尽提子
* 禁止全局同形再现
* 贴子，黑贴3又3/4子(7.5目)
* 行礼：谢谢指教
